03. Navigatie ()
Archi heeft een optie om navigatie snelkoppelingen toe te voegen maar mijn ervaring is dat je daardoor de weg door de views snel kwijtraakt. Wanneer je door via de model tree door de views navigeert raak je die weg niet zo snel kwijt. - Ik heb er daarom bewust voor gekozen om géén extra navigatie menus of opties toe te voegen.
